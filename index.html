Sanppy

Snappy Overview: In this project, an advanced bipedal robot with humanoid skeleton is built via manufactured aluminum skeleton with custom upgrades of servos, vision, quadra central processing, ASIC level machine learning, servo controller, and algorithms mostly in python. The robot provides high degrees of freedom which in this design it has seventeen servos as the keypoints of two-dimensional rotations. The embedded development board used in here provides a great set up communication protocols such as WIFI, BLE, USB, MIPI, and GPIOs. The four core CPU is embedded with Mendel flavor of Linux which provides many advanced features of operating systems such as remote connection, multi-tasking, networking, file management and etc. The vision incorporates a five mega pixel camera which is connected to the dev board via MIPI connection. Hence, providing high speed feed back to the CPU which is embedded onto the robot. The servos are wire-mapped to a servo controller which is mounted on the back of the robot and connected to the dev board via high-speed USB protocol. The control commands are sent from the dev board to the analog servo controller through USB protocol, and translated to servo control semantics. Given a reliable power supply, the robot is capable of performing many complex and advanced tasks such as walking or repeating gestures of the person in front.
